# Diagnóstico Cartoons — Julio 2026 (Wasabi)

Página web de diagnóstico de marca (Cartoons vs. Smart, Universo Binario, Miniso y Mosca):
redes sociales, mix de contenidos, web/UX, e-commerce, SEO, competencia y contenidos pautados en Meta.

## Contenido
- `index.html` — la página del diagnóstico (autocontenida: fuente Inter 18pt e imágenes embebidas).
- `Diagnostico-Cartoons-media/` — los videos (reels) que se reproducen dentro de la página.
- `.nojekyll` — evita el procesamiento Jekyll de GitHub Pages.

## Publicar con GitHub Pages
1. Creá un repositorio nuevo en GitHub (por ej. `cartoons-diagnostico-web`).
2. Subí **todo el contenido de esta carpeta** (index.html, la carpeta de media, README y .nojekyll) manteniendo la estructura tal cual.
3. En el repo: **Settings → Pages → Source: Deploy from a branch**, branch `main`, carpeta `/ (root)`. Guardá.
4. En unos minutos queda online en `https://<usuario>.github.io/cartoons-diagnostico-web/`.

> Mantené la carpeta `Diagnostico-Cartoons-media` junto a `index.html` para que los videos se reproduzcan.
